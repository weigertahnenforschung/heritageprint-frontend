Download Your Family Data

Extract structural profile data components quickly from standard historical search portals.

Export Steps for Ancestry.com

  1. Log into your database tree page dashboard on Ancestry.com.
  2. Open your tree drop menu array setup controls and click on Tree Settings.
  3. Locate the panel configuration option marked explicitly as Export Tree.
  4. Allow processing to finalize, then hit the link Download your GEDCOM file.

Export Steps for MyHeritage

  1. Open up your family workspace layout view on the primary site dashboard.
  2. Hover path maps directly over the secondary nav anchor item labeled Family Tree.
  3. Click on the specific submenu control layer line link item titled Manage Trees.
  4. Look across the configuration list and trigger the action titled Export to GEDCOM.

Export Steps for FamilySearch

  1. Because FamilySearch uses a single unified tree system structure, direct export files require specialized configurations.
  2. Access your profile dashboard parameters screen layout within the main interactive window interface view.
  3. Use an authorized platform synchronization extension app tool to extract structural generation data branches into a single .ged archive package.
